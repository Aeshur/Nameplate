# Nameplate

## Current Status

Maintained for newer Ashita v4 builds.

## Description

An Ashita v4 plugin that corrects FFXI's nameplate aspect ratio so nameplates stay sharp and the same size at any window resolution. It can also resize the damage font, hide Job Mastery stars, and set nameplate visibility modes.

## Usage

`/load Nameplate` to load it, then `/nameplate help` in game for the full command list.
